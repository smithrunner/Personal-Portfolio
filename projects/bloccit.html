Bloccit

Bloccit is a re-creation of the popular website Reddit, which is a message board where users have profiles and can create, comment, post and vote on any topic you could think of. I created Bloccit using Ruby on Rails and a MVC framework. I implemented each component of the application using TDD (Test-Driven-Development).

The main components of the application were a system for user authentication, allowing for various access levels tied to different logins and a voting system based on a time delay + point method.

The application was deployed using Heroku and can be accessed through this link: Bloccit. Feel free to create an account and play around on the site!
