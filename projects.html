Colorado School of Mines (August 2012- May 2016): Geophysical Engineering

Geophysical Engineering is a natural science concerned with the physical processes and properties of the Earth, analyized through quantitative methods. To put it concisely, a geophysicist is a combination of a computer scientist, geologist and physicist.

During my undergrad, every class was based around quantative data analysis and critical thinking. I took introductory programming classes in Python and Java, followed by extensive classes using MATLAB to process & interprate large amounts of data. The data used was often collected in the field by myself and fellow classmates. Survery types ranged from active source seismic surverys, to Electro-Magnetic surveys to gravity & magnetics surveys.

Bloc (June 2017 - Current): Software Development Program

Bloc is an online coding bootcamp lasting 8 months when working at the full time pace of 40 hours per week. The program has an emphasis on giving students real world coding experience through independent projects and mentorship. The program teaches Javascript, AngularJS, Ruby and Ruby-on-Rails and is divied into 6 Modules:

Module 1: Programming Fundamenetals (HTML, CSS and Git)
Module 2: Frontend Foundations (JS, DOM scripts)
Module 3: Backend Foundations (MVC framework, CRUD design, Ruby, Rails)
Module 4: Specialization (Building full stack web applications from scratch)
Module 5: Software Engineering Principles (data structures, algorithms, SQL, HTTP foundations)
Module 6: Open Source Apprenticeship (Work on open source projects)
